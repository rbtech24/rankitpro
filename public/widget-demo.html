How to Implement

Copy and paste this code into any website to add these widgets

Complete Implementation Code:
<!-- Step 1: Add the widget HTML where you want it to appear -->
<div 
  data-rankitpro-widget="checkIns"
  data-company-id="16"
  data-limit="6"
  data-columns="3"
  data-show-images="true"
  data-title="Recent Service Calls"
></div>

<!-- Step 2: Add the configuration script -->
<script>
window.RankItProConfig = {
  apiKey: 'b3e9eac681d470e6a925093552bc85f50b5b23541f0bdf82e4c838c9bc03cb51',
  endpoint: 'https://carrolltonsprinklerrepair.test',
  companyId: '16'
};
</script>

<!-- Step 3: Load the widget library -->
<script src="https://carrolltonsprinklerrepair.test/widget.js" async></script>

Widget Options:

  • data-rankitpro-widget: "checkIns", "blog", or "reviews"
  • data-company-id: Your company ID (16 for this demo)
  • data-limit: Number of items to display (3, 6, 9, 12)
  • data-columns: Grid columns (1, 2, 3, or 4)
  • data-show-images: "true" or "false" to show photos
  • data-title: Custom title for the widget

Works on Any Website:

These widgets work on WordPress, Wix, Squarespace, Shopify, custom HTML sites, and any platform that allows custom JavaScript. Simply paste the code and the widgets will automatically display your real business data.
