RankItPro Widget Examples for Non-WordPress Sites

These examples show how to embed RankItPro widgets on any website using simple HTML and JavaScript.

1. Service Calls Widget

Shows recent check-ins with photos, job details, and service documentation from your field team.

Basic Implementation

<!-- Include the RankItPro widget script -->
<script src="https://your-domain.com/widget.js"></script>

<!-- Widget container with data attributes -->
<div 
    data-rankitpro-widget="checkIns"
    data-company-id="YOUR_COMPANY_ID"
    data-limit="6"
    data-columns="3"
    data-show-images="true"
    data-title="Recent Service Calls"
></div>

Advanced Configuration

<script>
// Set global configuration
window.RankItProConfig = {
    apiKey: 'YOUR_API_KEY',
    endpoint: 'https://your-rankitpro-domain.com',
    companyId: 'YOUR_COMPANY_ID'
};
</script>
<script src="https://your-domain.com/widget.js"></script>

<!-- Multiple service call widgets with different settings -->
<div 
    data-rankitpro-widget="checkIns"
    data-limit="4"
    data-columns="2"
    data-show-images="true"
    data-title="Latest HVAC Services"
></div>

<div 
    data-rankitpro-widget="checkIns"
    data-limit="8"
    data-columns="4"
    data-show-images="false"
    data-title="All Recent Services"
></div>

Manual JavaScript Integration

<div id="service-calls-widget"></div>

<script>
// Manual widget initialization
window.RankItProConfig = {
    apiKey: 'YOUR_API_KEY',
    endpoint: 'https://your-rankitpro-domain.com'
};

// Load the widget script
const script = document.createElement('script');
script.src = 'https://your-domain.com/widget.js';
script.onload = function() {
    // Initialize service calls widget manually
    const container = document.getElementById('service-calls-widget');
    window.RankItPro.widgets.checkIns(container, {
        companyId: 'YOUR_COMPANY_ID',
        limit: 6,
        columns: 3,
        showImages: true,
        title: 'Our Recent Professional Services'
    });
};
document.head.appendChild(script);
</script>
Widget Preview: Service calls would display here with photos, job types, customer names, and service notes in a responsive grid layout.

Complete Website Integration Example

Here's how to integrate all three widgets on a single page:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Your Home Service Company</title>
</head>
<body>
    <!-- Your website header and navigation -->
    <header>
        <h1>Professional Home Services</h1>
        <nav>
            <!-- Your navigation menu -->
        </nav>
    </header>

    <main>
        <!-- Hero section -->
        <section class="hero">
            <h2>Expert HVAC, Plumbing & Electrical Services</h2>
            <p>Trusted by homeowners across the region</p>
        </section>

        <!-- Recent Service Calls Section -->
        <section class="services-section">
            <div 
                data-rankitpro-widget="checkIns"
                data-company-id="YOUR_COMPANY_ID"
                data-limit="6"
                data-columns="3"
                data-show-images="true"
                data-title="Our Recent Professional Work"
            ></div>
        </section>

        <!-- Customer Reviews Section -->
        <section class="testimonials-section">
            <div 
                data-rankitpro-widget="reviews"
                data-company-id="YOUR_COMPANY_ID"
                data-limit="6"
                data-columns="3"
                data-title="What Our Customers Are Saying"
            ></div>
        </section>

        <!-- Blog Posts Section -->
        <section class="blog-section">
            <div 
                data-rankitpro-widget="blog"
                data-company-id="YOUR_COMPANY_ID"
                data-limit="4"
                data-columns="2"
                data-show-images="true"
                data-title="Helpful Tips & Industry News"
            ></div>
        </section>
    </main>

    <!-- Footer -->
    <footer>
        <p>© 2025 Your Home Service Company</p>
    </footer>

    <!-- RankItPro Widget Configuration -->
    <script>
    window.RankItProConfig = {
        apiKey: 'YOUR_API_KEY',
        endpoint: 'https://your-rankitpro-domain.com',
        companyId: 'YOUR_COMPANY_ID'
    };
    </script>
    
    <!-- Load RankItPro widgets -->
    <script src="https://your-domain.com/widget.js"></script>
</body>
</html>

Widget Configuration Options

Service Calls Widget Options

  • data-company-id: Your unique company identifier (required)
  • data-limit: Number of service calls to display (default: 6)
  • data-columns: Grid columns (1, 2, 3, or 4, default: 3)
  • data-show-images: Show service photos (true/false, default: true)
  • data-title: Custom widget title

Blog Posts Widget Options

  • data-company-id: Your unique company identifier (required)
  • data-limit: Number of blog posts to display (default: 6)
  • data-columns: Grid columns (1, 2, 3, or 4, default: 2)
  • data-show-images: Show featured images (true/false, default: true)
  • data-title: Custom widget title

Reviews Widget Options

  • data-company-id: Your unique company identifier (required)
  • data-limit: Number of reviews to display (default: 6)
  • data-columns: Grid columns (1, 2, 3, or 4, default: 3)
  • data-title: Custom widget title

Setup Instructions

Important: Replace the following placeholders with your actual values:
  • YOUR_API_KEY: Your RankItPro API key
  • YOUR_COMPANY_ID: Your unique company identifier
  • https://your-domain.com: Your RankItPro domain

Step-by-Step Setup

  1. Get Your Credentials: Log into your RankItPro dashboard to find your API key and company ID
  2. Choose Widget Types: Select which widgets you want to display on your site
  3. Copy the Code: Use the examples above and customize the options
  4. Add to Your Website: Paste the code where you want the widgets to appear
  5. Test the Integration: Verify the widgets load correctly and display your data

Responsive Design

All widgets are fully responsive and will automatically adapt to different screen sizes:

  • Desktop: Full grid layout with specified columns
  • Tablet: Automatically reduces columns for better fit
  • Mobile: Single column layout for optimal mobile viewing
